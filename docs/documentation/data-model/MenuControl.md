<b style="font-size: 20px">(1) 작업그룹 추가그룹</b><br/>

수정하고자 하는 업무 Node를 선택한 후 마우스 오른쪽 버튼을 클릭하면 Context Menu가 활성됩니다. <br/>
활성 된 Menu에서 Rename를 선택하면 됩니다.
