# (RBK)

연결된 데이터베이스에 Rollback Command를 실행하는 컴포넌트입니다.

Remark

RBK BOC는 쿼리를 자동 생성하며 자동 생성에 필요한 속성을 설정합니다.

# 1. RBK BOC 이미지

BOC 컴포넌트 목록에서 RBK BOC를 선택하여 Drag-Drop으로 화면작업 영역에 디자인합니다.

# 2. RBK BOC 속성

화면작업 영역에서 RBK BOC 선택 시 속성 영역에 설정이 가능한 항목에 값을 입력합니다.

1) 1. 기초정보

① 논리명칭
BOC가 어떤 동작을 수행하는지를 간단한 명칭으로 기술합니다. 해당 명칭을 처리 흐름을 간략하게 표현할 수 있도록 기술합니다.

② 논리설명
논리명칭으로 표현하기 부족할 때 좀 더 상세히 내용을 기술합니다.

2) 3. 출력부

① Column Scroll
해당 BOC 목록이 많을 경우 "true"로 설정했을 때 Display Count에서 설정한 개수만큼 보여주고 나머지 목록은 ▲▼ 버튼으로 확인할 수 있습니다.

② Display Count
해당 BOC 목록이 많을 때 목록에 보이는 개수를 설정합니다.

3) 9. 기타

① 결과메시지
해당 처리 결과에 대한 메시지를 클라이언트. 즉, 호출자에게 보낼 필요가 있을 때 정의합니다.
