# 구성

1. Tool Box
Query Search, Add SQLEditor, Excute Query 등 POWER MDD에서 사용되는 각종 기능들을 제공합니다.

1) (Query Search)
Service Model에서 입력한 컬럼, 데이터명 등 쿼리를 검색합니다.

2) (Add SQLEditor)
Editor(작업창)가 하나 더 늘어납니다.

3) (Excute Query)
쿼리를 실행할 수 있습니다. 단축키(Ctrl + Enter)로도 실행이 가능합니다.
