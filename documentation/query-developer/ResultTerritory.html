# 결과 영역


1) 실행 확인
Completed(성공) Or Run error(오류)으로 실행을 확인할 수 있으며, 걸린 시간(1000 = 약 1초)도 같이 확인할 수 있습니다.

2) Data Grid
실행한 쿼리의 데이터가 출력됩니다.

3) Trace
실행한 쿼리 및 데이터 개수가 나옵니다.

4) Table
해당 테이블의 컬럼 정보 ,select, insert, update문을 보여줍니다.

5) Table Import
Data Grid에서 조회한 데이터 Row나 전체를 선택 후 오른쪽 마우스 클릭 후 Import를 선택하면 table import에 데이터가 나옵니다.
Data Load를 클릭하면 Insert나 Update을 할 수 있습니다.

6) A ~ H
Data Grid에서 조회한 데이터 Row나 전체를 선택 후 오른쪽 마우스 클릭 후 A ~ H를 선택하면 선택한 데이터 가 Copy되어서 나옵니다. 다른 테이블을 조회할 때 이전 데이터를 저장 후 사용하기 용이합니다.

7) Result Count, CurrentRow
실행된 쿼리의 개수와 현재 row 개수를 나타냅니다.

8) Clear
결과창을 Clear시킬 수 있습니다.
