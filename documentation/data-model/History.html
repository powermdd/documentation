# History

작업한 내역이 남는 영역입니다. Change Data에는 작업한 시간, User Id에는 작업한 유저명이 남습니다.


1) 원하는 내역을 두 번 클릭하면 해당 작업 시간대로 ERD가 새로고침됩니다.

>

Remark

History를 불러 왔을 경우 저장하지 않으면 가장 최근에 저장한 내역이 그대로 남습니다.
